# 财神供奉加成规则

## 核心机制：BUFF持续时间

| 供奉材料 | BUFF持续时间 |
|:---|:---|
| 📿 线香 | 1次化缘 |
| 🕯️ 蜡烛 | 3次化缘 |
| 📜 金纸 | 5次化缘 |
| 🍎 供果 | 10次化缘 |

## 各财神加成效果

| 财神 | 加成效果 | 说明 |
|:---|:---|:---|
| **赵公明** | 香火钱+20% | 供奉后，化缘获得香火钱×1.2 |
| **比干** | 法力上限+10 | 供奉后，法力上限临时+10（测试版暂不生效） |
| **关公** | 化缘收益+15% | 供奉后，化缘获得香火钱×1.15 |
| **范蠡** | 暴击率+5% | 供奉后，5%概率触发暴击（收益×3） |
| **李诡祖** | 供奉功德+10% | 供奉时功德+1.1（向下取整） |
| **沈万三** | 法力回复+20% | 供奉后，法力回复速度×1.2 |
| **刘海蟾** | 香火钱上限+1000 | 供奉后，香火钱上限临时+1000 |
| **子贡** | 每日首供双倍 | 每天第一次供奉收益×2 |
| **财神奶奶** | 化缘风险-10% | 供奉后，化缘负面事件概率-10% |

## 叠加规则

1. **同类型不叠加**：供奉同一个财神，刷新持续时间
2. **不同类型可叠加**：同时供奉赵公明+关公 = 收益+35%
3. **风险减免上限**：最高减免至5%（不能0风险）

## 举例

- 供奉赵公明（线香）→ 获得1次化缘+20%香火钱BUFF
- 供奉关公（蜡烛）→ 获得3次化缘+15%香火钱BUFF
- 同时供奉 → 1次化缘+35%，之后2次+15%
