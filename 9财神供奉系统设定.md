# 财神大陆 - 9财神供奉系统设定

## 供奉基础规则

### 1. 供奉位置
- **9个财神位**：只是选择界面，可同时供奉任意多个不同财神
- **没有位置限制**：9个都是可选的，不是"占坑"机制

### 2. 材料与持续时间

| 材料 | 消耗 | BUFF持续次数 | 供奉奖励 |
|:---|:---|:---|:---|
| 📿 线香 | 1个 | **1次**化缘 | +10香火钱 |
| 🕯️ 蜡烛 | 1个 | **3次**化缘 | +20香火钱 |
| 📜 金纸 | 1个 | **5次**化缘 | +30香火钱 |
| 🍎 供果 | 1个 | **10次**化缘 | +50香火钱 |

### 3. BUFF叠加规则

**不同财神 → 效果叠加，独立计时**
- 例：赵公明(5次) + 关公(3次) + 财神奶奶(10次)
- 化缘时：同时享受三种加成

**相同财神 → 刷新持续时间，不叠加**
- 例：先供奉关公5次，再供奉关公3次
- 结果：关公BUFF变成3次（取最新的）

## 9财神详细设定

### 1. 赵公明 - 正财神
- **效果**：化缘香火钱 **+20%**
- **类型**：化缘增益
- **上限**：无上限，可叠加其他财神
- **策略**：适合频繁化缘时使用

### 2. 比干 - 文财神
- **效果**：法力上限 **+10**（6级后生效）
- **类型**：被动增益
- **上限**：+10（固定值）
- **备注**：1-5级暂不生效，HTML测试版暂不实现

### 3. 关公 - 武财神
- **效果**：化缘收益 **+15%**
- **类型**：化缘增益
- **上限**：无上限
- **策略**：与赵公明同时供奉 = +35%收益

### 4. 范蠡 - 偏财神
- **效果**：化缘暴击率 **5%**（暴击时收益×3）
- **类型**：化缘增益
- **上限**：5%概率（独立计算）
- **策略**：赌运气，可能一夜暴富

### 5. 李诡祖 - 增福财神
- **效果**：供奉时功德 **+10%**（向下取整）
- **类型**：供奉增益
- **上限**：+10%（供奉时立即生效）
- **策略**：需要冲功德升级时供奉

### 6. 沈万三 - 禄财神
- **效果**：法力回复速度 **+20%**
- **类型**：持续BUFF
- **上限**：+20%（测试版1秒变1.2秒回复）
- **策略**：频繁化缘时保持供奉

### 7. 刘海蟾 - 寿财神
- **效果**：香火钱上限 **+1000**
- **类型**：持续BUFF
- **上限**：+1000（固定值）
- **策略**：防止存太多钱被大凶事件损失惨重

### 8. 子贡 - 喜财神
- **效果**：每日首次供奉 **双倍奖励**
- **类型**：每日增益
- **上限**：每天1次
- **备注**：HTML测试版暂不实现（无日期系统）

### 9. 财神奶奶 - 财神婆婆
- **效果**：化缘风险 **-10%**（最低5%）
- **类型**：化缘减益
- **上限**：风险最低降到5%，不能0风险
- **策略**：去高风险地区（北）必带

```
🙏 当前财神庇佑：赵公明(5)+钱 关公(3)+钱 财神奶奶(10)-险
```

### 优先级显示（空间不够时）
1. 化缘增益类（赵公明、关公）
2. 风险减免类（财神奶奶）
3. 持续BUFF类（沈万三、刘海蟾）

## 数值平衡公式

### 化缘最终收益计算
```
基础收益 = 地区基础值
× 事件倍数(大吉2.0/小吉1.5/顺利1.0/平淡0.6/小凶0.8)
× (1 + 赵公明加成0.2 + 关公加成0.15)
× 暴击倍数(3.0，如果触发)
= 最终收益
```

### 风险计算
```
实际风险 = 地区基础风险 - 财神奶奶减免
最低风险 = 5%（不会更低）
```

## 策略建议

### 新手期（1-5级）
- **供奉组合**：赵公明 + 关公（收益最大化）
- **材料选择**：蜡烛（3次，性价比高）
- **化缘地区**：东/南（低风险）

### 冲级期（攒功德）
- **供奉组合**：李诡祖（供奉时）+ 赵公明（化缘时）
- **材料选择**：金纸（5次，持续久）

### 高风险高回报
- **供奉组合**：财神奶奶 + 关公 + 范蠡
- **化缘地区**：北（40%→30%风险）
- **材料选择**：供果（10次，持久）

### 挂机积累
- **供奉组合**：沈万三 + 刘海蟾
- **策略**：让法力快速回复，上限提高，适合长时间游戏
