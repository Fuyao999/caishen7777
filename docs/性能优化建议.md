# 财神大陆 - 性能优化建议

## 客户端优化

### 1. 资源优化

**图片资源**
- 使用纹理压缩格式（ETC2/ASTC）
- 大图使用图集（Texture Packer）
- 动态加载资源，避免一次性加载过多
- 常用资源预加载，非常用资源按需加载

```typescript
// 动态加载示例
resources.load('images/background/level1', SpriteFrame, (err, spriteFrame) => {
    this.bgSprite.spriteFrame = spriteFrame;
});
```

**音效资源**
- BGM使用流媒体播放
- 短音效预加载到内存
- 控制同时播放的音效数量（最多8个）

### 2. 代码优化

**对象池使用**
```typescript
// 使用对象池复用节点
const nodePool = new NodePool();

// 获取节点
let node = nodePool.get();
if (!node) {
    node = instantiate(this.itemPrefab);
}

// 回收节点
nodePool.put(node);
```

**减少Draw Call**
- 使用图集减少材质切换
- 合理设置层级，避免不必要的重绘
- UI元素使用静态合批

**避免内存泄漏**
- 及时清理定时器
- 销毁场景时释放资源
- 避免循环引用

```typescript
onDestroy() {
    this.unscheduleAllCallbacks();
    // 释放资源
    resources.release('images/background/level1');
}
```

### 3. 渲染优化

- 限制帧率：30fps（非动作游戏足够）
- 使用遮罩裁剪，不渲染屏幕外元素
- 粒子效果控制数量（最多50个）
- 复杂动画使用骨骼动画代替帧动画

## 服务端优化

### 1. 数据库优化

**索引优化**
```sql
-- 常用查询添加索引
CREATE INDEX idx_user_id ON player_data(user_id);
CREATE INDEX idx_created_at ON login_records(created_at);
CREATE INDEX idx_status ON robots(status);
```

**查询优化**
- 避免SELECT *，只查询需要的字段
- 使用连接查询代替多次查询
- 分页查询，限制返回数量

```sql
-- 优化前
SELECT * FROM users WHERE level > 50;

-- 优化后
SELECT id, nickname, level FROM users 
WHERE level > 50 
ORDER BY level DESC 
LIMIT 20 OFFSET 0;
```

### 2. 缓存策略

**Redis缓存**
- 用户数据缓存（5分钟）
- 排行榜缓存（1小时）
- 配置数据缓存（长期）

```javascript
// 伪代码
const userData = await redis.get(`user:${userId}`);
if (userData) {
    return JSON.parse(userData);
}
const dbData = await db.query('SELECT * FROM player_data WHERE user_id = ?', [userId]);
await redis.setex(`user:${userId}`, 300, JSON.stringify(dbData));
return dbData;
```

### 3. 连接池优化

```javascript
// 数据库连接池配置
const pool = mysql.createPool({
    host: 'localhost',
    user: 'root',
    password: 'password',
    database: 'caishen_game',
    waitForConnections: true,
    connectionLimit: 20,      // 最大连接数
    queueLimit: 0
});
```

### 4. API优化

**响应压缩**
```javascript
const compression = require('compression');
app.use(compression());
```

**限流保护**
```javascript
const rateLimit = require('express-rate-limit');
const limiter = rateLimit({
    windowMs: 15 * 60 * 1000, // 15分钟
    max: 100 // 每个IP最多100请求
});
app.use('/api/', limiter);
```

## 网络优化

### 1. 数据包优化

- 使用二进制协议代替JSON（减少50%流量）
- 数据压缩传输
- 合并请求，减少HTTP连接数

### 2. 断线重连

```typescript
// 网络断开自动重连
private reconnectAttempts = 0;
private maxReconnectAttempts = 5;

onConnectionLost() {
    if (this.reconnectAttempts < this.maxReconnectAttempts) {
        setTimeout(() => {
            this.reconnectAttempts++;
            this.connect();
        }, 2000 * this.reconnectAttempts);
    }
}
```
