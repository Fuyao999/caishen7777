# 财神大陆 - 部署文档

## 1. 环境要求

- Node.js 18+
- MySQL 8.0
- Cocos Creator 3.8.0

## 2. 数据库初始化

```bash
# 登录MySQL
mysql -u root -p

# 执行建表脚本（密码：77889900）
source /Users/fuyao/Desktop/财神大陆-Cocos/database/schema.sql
```

## 3. 服务端部署

```bash
# 进入服务端目录
cd /Users/fuyao/Desktop/财神大陆-Cocos/server

# 安装依赖
npm install

# 启动服务器
npm start

# 服务器运行在 http://localhost:3001
```

## 4. 客户端运行

1. 打开 Cocos Creator 3.8.0
2. 打开项目：`/Users/fuyao/Desktop/财神大陆-Cocos/client`
3. 点击"预览"按钮

## 5. 测试账号

- 手机号：任意11位数字
- 验证码：1234

## 6. 游戏操作

1. 登录后进入主界面
2. 点击"供奉"进入庙宇，选择财神供奉
3. 点击"化缘"选择地区化缘
4. 点击"升级"查看进化条件
5. 点击"背包"查看物品
